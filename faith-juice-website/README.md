## 🍊 About

Faith is a 100% natural, preservative-free concentrated passion fruit drink handcrafted from the lush highlands of Kigezi, Uganda. One 750 ml bottle makes up to 4.5 litres of refreshing juice (dilute 1:5). UNBS certified. No artificial additives.

## 🚀 How to Host on GitHub Pages

1. **Create a GitHub account** at [github.com](https://github.com) if you don't have one.
2. **Create a new repository** — click the **+** icon → *New repository*.
   - Name it anything, e.g. `faith-juice` or `zoma-website`.
   - Set it to **Public**.
   - Do **not** initialise with a README (you already have one).
3. **Upload the files** — on the repository page click *uploading an existing file* and drag all three files into the box:
   - `index.html`
   - `.nojekyll`
   - `README.md`
4. **Enable GitHub Pages**:
   - Go to your repository → **Settings** → **Pages** (left sidebar).
   - Under *Source*, select **Deploy from a branch**.
   - Choose branch **main** and folder **/ (root)**.
   - Click **Save**.
5. **Wait 1–2 minutes**, then visit the URL shown — your site is live!
